# Venkey Portfolio

Premium animated portfolio website for Kuppili Leela Venkatesh, a Computer Science Engineering student at Pragati Engineering College.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP ScrollTrigger
- React Three Fiber
- Lenis
- Shadcn-style UI primitives

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verify

```bash
npm run typecheck
npm run build
```
